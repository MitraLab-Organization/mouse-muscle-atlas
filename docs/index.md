# Mouse Muscle Atlas (MMA)

An ontology for mouse muscle and skeletal anatomy structures.

## Overview

The Mouse Muscle Atlas (MMA) is a comprehensive ontology containing 173 anatomical structures including:

- **Bones**: Skull bones, ear ossicles, vertebrae, sternum, etc.
- **Muscles**: Facial muscles, pharyngeal muscles, laryngeal muscles, etc.
- **Cartilage**: Thyroid, arytenoid, cricoid cartilages, etc.

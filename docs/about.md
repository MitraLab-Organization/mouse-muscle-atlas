# About Mouse Muscle Atlas

## Description

The Mouse Muscle Atlas (MMA) is an ontology designed to provide a standardized vocabulary for mouse muscle and skeletal anatomy structures. It enables precise annotation and data integration across research studies.

## Scope

This ontology covers anatomical structures of the mouse head, neck, and upper body including:

### Skeletal Structures (Bones)
- Cranial bones (occipital, parietal, frontal, etc.)
- Facial bones (maxilla, mandible, nasal, etc.)
- Ear ossicles (malleus, incus, stapes)
- Vertebrae, sternum, clavicle, scapula

### Cartilaginous Structures
- Laryngeal cartilages (thyroid, cricoid, arytenoid)
- Alar cartilage

### Muscular Structures
- Facial expression muscles
- Mastication muscles
- Pharyngeal muscles
- Laryngeal muscles
- Hyoid muscles
- Tongue muscles

## Imports

MMA imports from the following ontologies:

- **RO** - Relations Ontology
- **BFO** - Basic Formal Ontology
- **PATO** - Phenotype and Trait Ontology
- **OMO** - OBO Metadata Ontology
- **UBERON** - Anatomy Ontology

## Development

MMA is developed using the [Ontology Development Kit (ODK)](https://github.com/INCATools/ontology-development-kit).

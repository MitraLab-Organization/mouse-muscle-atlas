# Mouse Muscle Atlas

Description: An ontology for mouse muscle and skeletal anatomy structures

More information can be found at http://obofoundry.org/ontology/mma

## Versions

### Stable release versions

The latest version of the ontology can always be found at:

http://purl.obolibrary.org/obo/mma.owl

(note this will not show up until the request has been approved by obofoundry.org)

### Editors' version

Editors of this ontology should use the edit version, [src/ontology/mma-edit.owl](src/ontology/mma-edit.owl)

## Contact

Please use this GitHub repository's [Issue tracker](https://github.com/MitraLab-Organization/mouse-muscle-atlas/issues) to request new terms/classes or report errors or specific concerns related to the ontology.
